# Alpha-1S-firmware-experiences
Experiences with  Alpha 1S motherboard and firmware


Board : Alpha 1S V1.1 Date 2015-01-04
MCU : STM32F103RDT6
Flash : 384 KB
RAM : 64 Kb
Debug natif : SWD + JTAG


You can set defaut servo positions at boot by creating new Default.HTS using Alpha 1S PC software ,select  export actions to HTS in right corner menu and rename it to " Default.hts" then put that on the robot SD card and go !
